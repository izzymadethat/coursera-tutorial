HTML is Only Structure

HTML is structure of a website.

It consists of elements and these elements either describe the type of content to display or tell the browser more information about the content.

Anatomy

Each element at its most basic form has:

  1. The element's name (like 'p' indicating the content is a paragraph section)
  2. The opening tag (< p >)
  3. The content after the opening tag, which is the text inside the paragraph
  4. Finally, the closing tag (< / p>)

Not everything has to have closing tags though.
< br> (Line break) and < hr > (Horizontal rule) don't need closing tags. They are considered self-closing

Each element can have its own attributes. We use these to classify certain elements or add specific styling to the elements.

id: id attributes are single-use
class: classes can be used to classsify a group of elements that will share the same styling and attributes
functions: each elements can have functions like onclick, which tells the browser what to do when an element is clicked

Everything in html is sequential, from top to bottom.


HTML Content Models

Block

Block elements render on a new line by default.

Common Block Elements

  • div tag
  • h1-h6 text
  • p text
  • all list elements
  • all table elements
  • nav tag

Inline Elements

Inline elements stay on the same line by default.

Common Block Elements

  • a (links)
  • img
  • button
  • span
  • script
  • input
